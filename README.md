## Static Website Hosting Using AWS

This project shows how to effectively use Amazon S3 and Amazon CloudFront to deploy and serve a static website.
It was finished as a component of the Accenture Primers Cloud Project, which sought to provide practical experience in the deployment of websites using AWS services.
